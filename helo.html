Quick deploy

  1. Push this file to a Git repo (or keep locally).
  2. For GitHub Pages: put file in root and enable Pages in repo settings.
  3. For Netlify: drag & drop into Netlify "Sites" or connect Git and deploy.
  4. For Vercel: run vercel in the folder (needs Vercel CLI) or connect Git repo.
